## 1. Scope 与稿件定位

- 期刊接收 original research、methods、reviews、perspectives 等类型。
- 期刊强调两类主线：
  - 用数学科学工具得到新的 biological insights；
  - 具有明确 biological applicability 的新数学/统计/计算方法。
- 当前 `paper1` 更适合按 `Original research article` 来组织；如果后续更强调方法学本身，也可评估 `Methods` 口径。

## 2. 审稿与出版模式

- 审稿：`single-blind peer review`
- 投稿系统：Editorial Manager
- 出版模式：`hybrid`
- 订阅路线：可投，`no APC charges apply`
- OA APC（当前官网标价）：`£2290 / $3190 / €2590`

## 3. 主稿硬要求

- title 要 concise and informative
- title page 需要：
  - 作者姓名
  - 作者单位
  - 通讯作者邮箱
  - ORCID（如果有）
- abstract：`150-250` 词
- keywords：`4-6`
- 需要 `Statements and Declarations`
- Acknowledgments / grants / funds 应放在 title page 的单独部分
- 正式提交版需要连续行号和连续页码

## 4. 正文与参考文献格式

- LaTeX 投稿受鼓励，Word 也接受
- 需要提交可编辑源文件；缺少 editable source files 可能直接不送审
- 标题层级使用 decimal system，最多三级
- 缩写首次出现必须定义
- 文内引用：`name and year in parentheses`
- 参考文献表：不加数字序号，按第一作者字母顺序排序
- DOI 尽量写成完整链接

## 5. 图表要求

- figure 文件电子提交
- 优选格式：
  - vector: `EPS`
  - halftone: `TIFF`
  - MS Office 文件也接受
- 分辨率：
  - line art: `1200 dpi`
  - halftone: `300 dpi`
  - combination art: `600 dpi`
- 彩图在线版免费
- caption 放在正文文件中，不放进图文件里

## 6. 补充材料要求

- Supplementary Information 单独提交
- 每个 SI 文件应包含：
  - article title
  - journal name
  - author names
  - affiliation
  - corresponding author e-mail
- 文本/展示类补充建议用 `PDF`
- 表格类可用 `.csv` 或 `.xlsx`
- supplementary files 将基本按收到时原样发布
- 正文中应明确引用 supplementary material；Springer 建议口径为 `Online Resource`

## 7. 数据与声明

- original research 必须包含 `Data Availability Statement`
- 期刊强烈鼓励把 supporting data 放到公开仓库
- competing interests 需要明确写出；没有也要写 `none` 或等价表述
- 生成式 AI 不能列为作者；如果用于实质性内容生成，应在 Methods 或合适位置说明

## 8. 对 paper1 的直接影响

- 现在最重要的不是继续压页数，而是把 submission package 结构做对：
  - 主稿作者-年份引用
  - 4-6 个关键词
  - title page / declarations / cover letter 对齐
  - supplementary file 与 Springer 口径对齐
- 当前英文摘要自动审计为 `232` 词，已经落在要求范围内。
- 当前 figures 已有 `PNG/TIFF` 双版本，分辨率路线基本够用。
